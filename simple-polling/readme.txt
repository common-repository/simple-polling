Add ajax based simple polling/election system in your WordPress post or widget.
== Description ==

Using this plugin you can add a polling/election on your WordPress website

*	In the Blog post 
*	In a Page
*	In a Widget


Simple and ajax based polling system with the result in percentage and with total votes. You can use two images for polling/election with the name and polling title. User can vote using ajax. This is a cookie based election/polling system so user can participate only one time.



== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Click on the 'Setting' option and then click on the 'Simple Polling'.
